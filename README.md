# Crystal Collectors

Tripp Stringfield

In Crystal Collectors, you must collect the appropriate amount of crystals without going over.
Collecting too many crystals will result in a loss

Written with HTML5, CSS3, Javascript, JQuery

The 4 crystals will all have different values for each game.
Other game instructions provided on page.

This game is deployed at: https://jastring1.github.io/unit-4-game
